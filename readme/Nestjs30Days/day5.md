   > 管道通常有两种作用：
   >
   > - **转换/变形**：转换输入数据为目标格式
   > - **验证**：对输入数据时行验证，如果合法让数据通过管道，否则抛出异常
   >
   > 管道 pipe 和过滤器 filter 之间的关系：管道偏向于服务端控制器逻辑，过滤器则更适合用客户端逻辑
